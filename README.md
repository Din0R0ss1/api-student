# API Student Laravel

REST API sederhana untuk mengelola data student menggunakan Laravel.

## Fitur
- GET semua data student
- GET berdasarkan ID
- POST (tambah data)
- PUT (update data)
- DELETE (hapus data)

## Tools
- Laravel
- MySQL
- Postman

## Cara Menjalankan
1. Clone repository
2. Jalankan `php artisan migrate`
3. Jalankan `php artisan serve`
